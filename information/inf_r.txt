1.Общие сведения
Программа weapon.exe предназначена для демонстрации знаний, приобретенных обучаемым
в области технологии разработки объектно-ориентированных программ средствами языка Си++.
Для работы программы необходима ОС Windows

2.Функциональное назначение
Данная программа моделирует работу тира. Функционал программы:
Создание тира;
Добавление/удаление посетителей;
Добавление/удаление оружия;
Добавление патронов;
Изменение характеристик каждого объекта;

3.Используемые технические средства
ЭВМ на основе архитектуры х86-64 с наличием монитора и клавиатуры.

4.Вызов и загрузка
Запуск программы осуществляется через запуск файла weapon.exe
 
5.Входные данные
Входные данные - текстовой файл конфигурации, файлы со служебной информацией, ввод с клавиатуры.

6.Выходные данные
Выходные данные – стандартный поток вывода и файловый поток (файл сохранения данных).
