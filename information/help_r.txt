1.Назначение программы
Программа предназначения для использования в высших учебных заведения с целью демонстрации знаний,
приобретенных обучаемым в области технологии разработки объектно-ориентированных программ средствами языка Си++.

2.Условия выполнения программы
Устройство ввода (клавиатура)
Операционная система – семейство ОС Windows

3.Выполнение программы
Запуск программы осуществляется через запуск файла weapon.exe

4.Сообщения оператору
1)«Файл не открыт!» - Неверно указанный путь к файлу локализации в config.txt или файл с таким названием отсутствует.
2)«В языковом файле лишние строки! Работа программы будет прекращена!» / «В языковом файле недостаточно строк! Работа
программы будет прекращена!» - файл локализации не соответствует требованиям, верните исходное содержимое (263 строки).
3)«Отсутствует файл конфигурации! Работа программы прекращена!» - Файл конфигурации config.txt не был найден в
папке с программой.
4)«В файле конфигурации лишнии строки! Работа программы прекращена!» / «В файле конфигурации недостаточно строк!
Работа программы прекращена!» - файл конфигурации config.txt не соответствует требованиям, верните
исходное содержимое (12 строк).
5)«Файл с информацией об объектах не открыт!» - файл с информацией для создания объектов не был найден,
проверьте путь к файлу и имя.
6)«В файле объектов присутствуют лишние строки! Дальнейшее создание объектов из файлов невозможно» / «В файле объектов
недостаточно строк! Дальнейшее создание объектов из файлов невозможно» - файл не соответствует требованиям,
верните исходное содержимое  (6 строк).
7)«Ошибка в данных файла для создания объектов! Объект не будет создан!» - файл объектов не соответствует
требованиям, верните исходное содержимое или проверьте правильность введенных в него данных:
1-я строка предназначена для создания тира и посетителя: *время работы тира (до 8 часов)* *адрес тира* *имя посетителя*
*возраст посетителя* *карта посетителя: 1-завести, 0-не заводить* = 5 слов в строке.
2-я строка предназначена для создания гранат: *количество гранат (до 10)* *радиус поражения гранаты
(до 7)* = 2 слова в строке.
3-я строка предназначена для создания пистолета: *размер магазина (до 10)* *калибр (до 12.7)* *прицельная дальность
(до 50)* *четырехзначный номер пистолета* *тип ствола: 1-Гладкоствольный, 2-Нарезной* = 5 слов в строке.
4-я строка предназначена для создания автомата: *размер магазина (до 45)* *калибр (до 9)* *прицельная дальность (до 1000)*
*четырехзначный номер автомата* *тип ствола: 1-Гладкоствольный, 2-Нарезной* = 5 слов в строке.
5-я строка предназначена для создания еще одного посетителя:  *имя* *возраст* * карта посетителя: 1-завести,
0-не заводить* = 3 слова в строке.
